Cloud Hopper

Soar high into the clouds with Cloud Hopper, an addictive endless jumping game. Each cloud is a puzzle on its own - sometimes straight forward, but often tricky. Take control of a happy lightning bug as you hop to new heights!



Cloud Hopper is an Android game that a friend and I created in our spare time. The goal of this project was to gain experience in designing and developing a game for the the Android ecosystem. LibGDX was used as the game platform and Java as the progamming language. The developement was done in Android Studio.

UMN Schedule Export

A Chrome Extension that allows University of Minnesota students to export their class schedule to Google Calendar or iCal.



UMN Schedule Export is a project that I started to help reduce the effort required for University of Minnesota students to add their class schedules to common calendar applications like iCal and Google Calendar. After semesters of manually creating the class events in Google Calendar, I decided there needed to be a way to export the class data so it could be easily imported to calendar apps. UMN Schedule was the result. It was built using Javascript, jQuery and HTML.

Resistance and Avalon

This app lets you play Resistance, along with its expansion game, Avalon, on your phone. Resistance is a party game for 5-10 players. It is a social deduction game.



Resistance and Avalon is an Android game that a friend and I created to solve some of the issues we had with the original board game. The original board game had no safeguards against players beign able to cheat. It also can be cumbersome to carry around a board game and all the pieces required to play it. Our Android application was designed to prevent cheating and allow for smoother gameplay. Simialar to Cloud Hopper, we used LibGDX as the game platform and Java as the progamming language. The developement was done in Android Studio.
